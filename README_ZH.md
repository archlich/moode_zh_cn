# moode_zh_cn([Eng](/README.md))
MoOde 音乐播放器的中文化项目

## 如何使用

通过SSH 连接到Pi (用户名:pi 密码:moodeaudio) 然后运行以下代码。
```
cd /home/pi
git clone https://github.com/jerrykuku/moode_zh_cn.git
cd moode_zh_cn
./install.sh
```
脚本执行完成后，系统将会立刻重启，启动完成后即可进入 系统设置页面 切换语言。
